# Theatre du labyrinthe

Site internet de la compagnie du `Théâtre du labyrinthe`.

# Utilisation local

```bash
python3 -m venv venv
source venv/bin/activate
python3 -m venv env
source env/bin/activate
pip install python-lsp-server jinja2
make clean
make
```
