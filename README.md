# Theatre du labyrinthe

Site internet de la compagnie du `Théâtre du labyrinthe`.
> [lien du site](https://www.theatredulabyrinthe.com)

# Utilisation local

```bash
python3 -m venv venv
. venv/bin/activate
pip install flask
export FLASK_APP=run
flask run
```

Aller à `http://127.0.0.1:5000`.
